# AutoMateX

**AutoMateX** is a Web-Based Vehicle Service and Maintenance Management System tailored for the individual vehicle owner in Sri Lanka. It serves as a unified digital platform addressing the fragmentation of personal vehicle health data.

## Project Overview

The system offers a "Digital Vehicle Passport," moving away from archaic, easily-lost paper service records. By centralizing service history, tracking fuel efficiency, and automating maintenance reminders, AutoMateX aims to prevent reactive maintenance and reduce the "Knowledge Gap" concerning spare parts.

### Key Features

*   **Vehicle Lifecycle Management:** Register and manage multiple vehicles, complete with a comprehensive history log of repairs, services, and fuel entries.
*   **QR Identity:** Unique QR code generation for each vehicle, linking to a read-only public version of its service history.
*   **Intelligent Suggestion Engine:** A rule-based algorithm that predicts the "Next Service Due" date using the vehicle's "Last Service Mileage" and calculated "Average Daily Mileage".
*   **Parts Verification Database:** Empowers users to verify OEM part numbers against a curated technical database (pilot phase targeting 5-10 common models).
*   **Garage Interface & Booking System:** A dedicated dashboard for partnered mechanics to manage service booking requests and issue digital PDF invoices directly to the user's history log.
*   **Manual Service Logger:** Flexibility for DIY repairs or services done outside the partner network, with receipt image upload support.

## Tech Stack

*   **Backend:** Laravel (PHP)
*   **Frontend:** HTML, CSS, JavaScript, Bootstrap 5 (Progressive Web App support)
*   **Database:** MySQL
